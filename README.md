# Mobius MCMS

Mobius Change Management System manages platform change propagation, repository synchronization, and approval records.

## Mission

Ensure approved changes are carried across affected repositories, diagrams, manifests, memory records, and metrics.

## Core Responsibilities

- Platform Change Proposals
- Repository impact analysis
- Sync status tracking
- Architecture update tracking
- Diagram update tracking
- Memory event coordination
- Pulse metric coordination
- Pending work reports

## Rule

A change is not complete until affected repositories are synchronized or explicitly marked pending.
